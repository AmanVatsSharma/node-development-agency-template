# SEO Checklist for Landing Pages

## Pre-Launch Checklist

### Metadata ✅
- [ ] `metadata.ts` file created
- [ ] Title tag (60-70 chars, keyword-rich)
- [ ] Meta description (150-160 chars, compelling)
- [ ] Keywords array (10-15 relevant keywords)
- [ ] Canonical URL set
- [ ] Open Graph title
- [ ] Open Graph description
- [ ] Open Graph image (1200x630px)
- [ ] Twitter Card title
- [ ] Twitter Card description
- [ ] Twitter Card image
- [ ] Robots directives (index/follow)

### Structured Data (JSON-LD) ✅
- [ ] Service Schema (for service pages)
- [ ] FAQ Schema (if FAQ section exists)
- [ ] Breadcrumb Schema
- [ ] Organization Schema (global)
- [ ] Review Schema (if testimonials exist)
- [ ] Structured data validates (use Google Rich Results Test)

### Sitemap ✅
- [ ] Page added to `sitemap.ts`
- [ ] Priority set (0.5-1.0)
- [ ] Change frequency set
- [ ] Last modified date set

### Content Optimization ✅
- [ ] One H1 tag (keyword-rich)
- [ ] Proper H2-H6 hierarchy
- [ ] Keywords in first 100 words
- [ ] Internal links to related pages (3-5 links)
- [ ] External links (authoritative sources)
- [ ] Image alt text (descriptive)
- [ ] Content length (1500+ words for service pages)

### Technical SEO ✅
- [ ] Page loads in < 3 seconds
- [ ] Mobile responsive
- [ ] No broken links
- [ ] Proper URL structure (lowercase, hyphens)
- [ ] HTTPS enabled
- [ ] No duplicate content
- [ ] Proper redirects (301 for moved pages)

### Images ✅
- [ ] All images have alt text
- [ ] Images optimized (WebP format)
- [ ] Image file sizes < 200KB
- [ ] Descriptive file names
- [ ] Open Graph image exists

### Performance ✅
- [ ] Core Web Vitals pass
- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1
- [ ] Images lazy loaded
- [ ] JavaScript minified

## Post-Launch Checklist

### Verification ✅
- [ ] Page indexed in Google (check Search Console)
- [ ] Structured data appears in search results
- [ ] Page appears in sitemap.xml
- [ ] Robots.txt allows indexing
- [ ] No crawl errors in Search Console

### Monitoring ✅
- [ ] Google Analytics tracking
- [ ] Google Search Console setup
- [ ] Monitor rankings for target keywords
- [ ] Track organic traffic
- [ ] Monitor conversion rates

## Monthly Review Checklist

### Content Updates ✅
- [ ] Review and update content
- [ ] Add new internal links
- [ ] Update statistics/metrics
- [ ] Refresh testimonials

### Technical Review ✅
- [ ] Check page speed
- [ ] Verify all links work
- [ ] Review mobile experience
- [ ] Check for broken images

### SEO Performance ✅
- [ ] Review Search Console data
- [ ] Check keyword rankings
- [ ] Analyze organic traffic
- [ ] Review conversion rates
- [ ] Update sitemap if needed

## Quick Reference

### Priority Levels
- **1.0** - Homepage
- **0.9** - Main service pages
- **0.8** - Sub-service pages
- **0.7** - Blog posts
- **0.5** - Legal pages

### Change Frequencies
- **daily** - Blog, News
- **weekly** - Service pages, Portfolio
- **monthly** - About, Services
- **yearly** - Legal pages

### Image Specifications
- **OG Image:** 1200x630px
- **Twitter Card:** 1200x675px
- **Format:** WebP or JPG
- **Max Size:** 200KB
