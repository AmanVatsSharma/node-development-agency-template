React.js is the most popular frontend framework in the world, and India has thousands of "React developers." Quality ranges from world-class to disastrous. Here's how to hire the right partner.

## What Good React.js Companies Actually Deliver

- TypeScript-first (not plain JS)
- Next.js or Vite for modern tooling (never CRA)
- Proper state management (Zustand, Redux Toolkit, or React Query)
- Component libraries (Tailwind + shadcn, or Material UI)
- Accessible, semantic HTML
- Core Web Vitals scores above 90
- Automated testing (Jest + Testing Library)
- CI/CD pipelines

## Red Flags

1. **Portfolio screenshots only, no live URLs** — can't verify quality
2. **Uses Create React App in 2025** — outdated, unmaintained
3. **No TypeScript** — guaranteed maintenance nightmare
4. **No Lighthouse scores to share** — they don't measure performance
5. **No testing** — quality gamble
6. **Mentions "React Native is the same as React"** — it's not
7. **Flat hourly rates without scoped deliverables**

## The Technical Questions to Ask

1. What version of React do you default to? (Should be 19+)
2. Next.js or Vite? (Both acceptable, depends on SEO needs)
3. How do you handle form validation? (React Hook Form + Zod is the modern default)
4. How do you handle API state? (React Query / TanStack Query is standard)
5. What's your approach to accessibility?
6. How do you measure and optimize Core Web Vitals?
7. Do you write tests? What's your coverage target?
8. How do you handle environment variables and secrets?
9. What's your deployment pipeline?

A good React company answers all of these with specifics.

## The "Next.js or Plain React?" Question

For 95% of projects, **Next.js is the correct answer**. Plain React with Vite is acceptable for:
- Internal dashboards (no SEO needed)
- Embedded widgets
- Chrome extensions
- Electron apps

For everything else — marketing sites, e-commerce, SaaS, blogs — Next.js wins on SEO, performance, and maintainability.

## Frequently Asked Questions

**How much does a React developer cost per month in India?**
Senior: ₹1,20,000–₹2,50,000/month. Mid-level: ₹60,000–₹1,20,000/month.

**How long does it take to build a React application?**
Depends on scope — simple landing page: 1–2 weeks. Full SaaS product: 3–6 months.

**Should I use Next.js or Create React App?**
Never CRA (unmaintained). Use Next.js for SEO-sensitive sites, Vite for dashboards.

**What's the difference between React.js and React Native?**
React.js is for web. React Native is for mobile (iOS/Android). Different skills, partially overlapping knowledge.
