## 🎨 Key Design Elements

### Animations
- **Counter Animation**: 1.5s with easeOutQuart easing
- **Stat Cards**: Staggered appearance (0.7s + index * 0.1s)
- **Hover Effects**: Scale 1.05, lift -8px, glow
- **Transitions**: All 0.3-0.6s duration

### Gradients
```css
Primary CTA:    from-amber-500 via-yellow-500 to-amber-600
Backgrounds:    from-slate-950 via-slate-900 to-amber-950
Founder Card:   from-blue-600 via-purple-600 to-pink-600
Text Gradients: from-amber-200 via-yellow-100 to-amber-200
```

### Shadows
```css
Cards:      shadow-xl
Important:  shadow-2xl
CTA Buttons: shadow-2xl shadow-amber-500/50
```
